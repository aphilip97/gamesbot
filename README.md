# gamesbot

> **NOTE** : This is a project for learning how to make Discord bots.

## DEV Setup

```sh
npm run build # Compiles TS into JS
npm run build:watch # Recompiles TS into JS if changes were made
npm run start # Starts the bot
npm run dev # Restarts the bot if changes were made
npm run test # tests the bot
npm run test:watch # doesn't stop testing the bot
npm run prepare # Ask the husky package what this does
```
